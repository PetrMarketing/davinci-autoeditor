# AutoEditor — DaVinci Resolve Plugin

## Язык
Весь код, комментарии, docstring'и, сообщения пользователю, логи и UI-тексты должны быть на **русском языке**. Исключение — имена переменных, функций и технические термины API.

## Структура проекта
- `main.py` — точка входа (запуск из меню Resolve Scripts)
- `config.py` — настройки, сохранение в JSON
- `ui/main_window.py` — интерфейс на UIManager (Qt встроенный в Resolve)
- `core/` — 10 модулей по шагам автомонтажа
- `utils/` — вспомогательные утилиты (SRT, таймкоды, логгер)
- `assets/` — ресурсы (переходы, фоны, стили титров)

## Архитектура
- Плагин для DaVinci Resolve Studio (Python Scripting API)
- Внешние зависимости: `pydub`, `httpx`, `ffmpeg`
- AI обработка через OpenRouter API
- 10 шагов: импорт → синхр. аудио → удаление тишины → субтитры → AI чистка → нарезка → мультикам → зум → переходы → титры

## Соглашения
- Все пользовательские сообщения и логи на русском
- Имена файлов и функций на английском (snake_case)
- Модули импортируются локально внутри функций (ленивый импорт) для совместимости со средой Resolve
- Конфигурация хранится в `autoeditor_config.json` рядом со скриптом
