Как это работает

Три простых шага — от скачивания до готового результата

1

Скачай плагин

Загрузи ZIP-архив и распакуй в удобное место на диске

2

Открой в Resolve

Запусти скрипт через меню Workspace → Scripts в DaVinci Resolve

3

Нажми «Запустить все»

Выбери нужные шаги или запусти все 10 — автомонтаж сделает остальное

Требования

Убедитесь, что у вас установлены все необходимые компоненты

DaVinci Resolve Studio 18+

Требуется Studio-версия для доступа к Scripting API

Python 3.10+

Для запуска скриптов автоматизации и AI-модулей

ffmpeg

Для анализа аудиодорожек, обнаружения тишины и обработки медиа

API-ключ OpenRouter

Для AI-функций: очистка речи, генерация субтитров и титров

Установка

Выберите вашу операционную систему и следуйте инструкциям

# 1. Установите Python и ffmpeg через Homebrew brew install python@3.10 ffmpeg # 2. Распакуйте архив unzip AutoEditor.zip -d ~/AutoEditor # 3. Установите зависимости cd ~/AutoEditor pip3 install -r requirements.txt # 4. Укажите API-ключ cp config.example.py config.py nano config.py # Вставьте ваш OpenRouter API-ключ

Скрипт будет доступен в Resolve: Workspace → Scripts → AutoEditor
